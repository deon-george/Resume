Welcome to the repository for my professional resume, written in LaTeX. This project serves as a centralized, version-controlled source for my resume, allowing for clean, professional formatting and easy updates.

## 🚀 How to Build

To compile the PDF from the LaTeX source code, you will need a LaTeX distribution installed on your system (such as TeX Live, MiKTeX, or MacTeX).

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/deon-george/Resume.git
    cd Resume
    ```

2.  **Compile the document:**
    You can compile the `.tex` file using `pdflatex`:
    ```bash
    pdflatex deonGeorge.tex
    ```
    This will generate a `deonGeorge.pdf` file in the same directory.

*(Alternatively, you can use online editors like [Overleaf](https://www.overleaf.com/) by simply uploading the `.tex` file.)*
